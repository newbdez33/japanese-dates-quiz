# 日语月份・日期 读法练习 / Japanese Month & Date Reading Trainer

一个纯前端的翻卡练习工具，帮助记忆日语里容易混淆的三套读法：

- **〜月（がつ）** — 日历月份（一月〜十二月），含 四月 しがつ・七月 しちがつ・九月 くがつ 等不规则读法
- **〜ヶ月（かげつ）** — 时长 / 几个月（一ヶ月〜十二ヶ月），含促音变化，且 四/七/九 读作 よん/なな/きゅう（与月份相反）
- **〜日** — 日期 / 几号（一日〜三十一日），含 ついたち・はつか・じゅうよっか 等特殊训读

## 功能

- 翻卡自评：看汉字 → 翻面看 假名 + 罗马音 + 🔊 朗读发音 → 标记「记住了 / 还不会」
- 进度自动保存在浏览器本地（localStorage），下次打开继续
- 「只练还不会的」筛选、罗马音显隐开关、重置进度
- 键盘操作：`空格` 翻面，`→/2` 记住了，`←/1` 还不会，`s` 朗读，`d` 切换卡组
- 手机 / 桌面自适应，可离线使用

## 使用

直接打开 [`index.html`](index.html) 即可，无需安装或构建。
